# Akudemy Frontend

This is the standalone Next.js frontend app for Akudemy.

## Features
- Landing page for Akudemy
- Placeholder for user dashboard
- Built with Next.js 14, React 18

## Getting Started

### Prerequisites
- Node.js 20+

### Development
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Build & Production
```bash
npm run build
npm start
```

## Linting
```bash
npm run lint
```
